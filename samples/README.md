# Samples

This directory contains sample images and example projects for testing Sprijt.

## Images

- **red-square.png** - Solid red 400x400 square (good for color replace testing)
- **gradient.png** - Purple-blue gradient (good for testing color operations)
- **checkerboard.png** - Blue and white checkerboard pattern
- **color-blocks.png** - Four colored blocks (red, green, blue, yellow)

## Usage

Use these images with the Load Image node to test your image processing workflows.

## Example Workflow

1. Load Image → red-square.png
2. Color Replace → Change red to blue
3. Preview → See the result
4. Save Image → Export the processed image
